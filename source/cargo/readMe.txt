This is the supplemental material for Sec. 3.5: The Cargo-Loading Problem. 

You can build the program by calling CMake to generate the build script, then calling Make to build the target executable, cargo.out. 

The main() function is in cargo.c. As you can see from its preamble, this program was written to verify the results on pp. 28-29 and p. 31 of Applied Dynamic Programming; and I have chosen to hardcode the particular example of interest. To check the output for different inputs, you can change CAPACITY, VALUES, or WEIGHTS, and rebuild the program.
